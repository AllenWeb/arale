
# Overlay

对话框组件，提供对话框显示隐藏、dom结构自定义、定位、select遮挡、确定取消关闭等功能特性。

继承了arale.overlay组件。

---


## 模块依赖

 - [seajs](seajs/README.md)
 - [jquery](jquery/README.md)
 - [overlay](overlay/README.md)


## API说明

* `trigger` : {element}

    对话框触发点。

* `comfirmElement` : {element}

    确定或提交按钮。

* `closeElement` : {element|array}

    关闭按钮，可以为节点数组。

* `onComfirm` : {function}

    确定时的操作。

* `onClose` : {function}

    关闭时的操作。
