associate 将两个数组合并成key:value Object，传入的数组的元素作为key

clean 清除所有未定义(undefined)的变量并返回新数组。

combine 合并两个数组，去掉重复项。

contains 是否包含提供的项。

each 遍历数组(对象)。此方法不允许通过返回值停止遍历。如果需要可以中断的遍历，请使用every, some方法。

every 测试是否数组(对象)每一项符合所给条件。当callback返回false时立刻停止遍历，并且every函数的调用返回值为false。

some 遍历数组，只要有一项满足条件则返回true，并且停止遍历。

filter 过滤出与条件相符的项并返回新数组(对象)。

map 改变每一项并创建一个新数组。

更多方法请参考API文档。
