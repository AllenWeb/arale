Overview
========

`arale.hash`Hash扩展模块，封装对象返回增强Hash对象，并提供each，sort等健值操作的方法。

**`arale.hash`是arale core的一部分**


Usage
======

源代码示例
    //通过set设置key、value 
    H({}).set('name','xx'); //return{'name':'xx'}
API
=======

*   each()
    遍历hash

*   set()
    设置key value
    
    更多API请参考API文档。
